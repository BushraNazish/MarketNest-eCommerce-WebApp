# MarketNest eCommerce Web App

Welcome to the MarketNest eCommerce Web App!

## Overview
MarketNest is a full-stack e-commerce solution separated into functional frontend and backend modules.

### Structure
- `marketplace-backend/`: Contains the backend API, logic, and database integrations.
- `marketplace-frontend/`: Contains the user interface and frontend application.

## Getting Started

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Running the Backend
1. Navigate to the backend directory:
   ```bash
   cd marketplace-backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the backend server:
   ```bash
   npm start
   ```

### Running the Frontend
1. Navigate to the frontend directory:
   ```bash
   cd marketplace-frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend development server:
   ```bash
   npm start
   ```
